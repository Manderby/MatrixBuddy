Description:
------------
Matrix Buddy is an application running and macOS, capable of computing
vector and matrix operations in 2D, 3D and 4D.



Cloning with submodules:
------------------------
Matrix Buddy requires the the submodule "NALib".

1. Either clone Matrix Buddy with the following git command:
   git clone --recursive https://github.com/Manderby/MatrixBuddy.git

2. Or write the following commands:
   git clone https://github.com/Manderby/MatrixBuddy.git
   cd MatrixBuddy
   git submodule init
   git submodule update

3. Open the provided project/solution file in the MatrixBuddy/proj folder.
   It should compile and run without errors in XCode.
